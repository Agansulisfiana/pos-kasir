Daily Kopi ToGo - Aplikasi POS

Fitur:
- Dashboard ringkas seperti aplikasi POS
- Top 5 produk terlaris hari ini
- Grafik penjualan 7 hari terakhir
- Alert stok menipis
- Tambah produk
- Harga modal
- Harga jual
- Format titik ribuan otomatis pada setiap input nominal uang
- Margin/laba otomatis
- Kategori produk
- Stok awal, terjual, stok akhir
- Keranjang belanja
- Layout Kasir dua kolom dengan scroll internal seperti POS profesional
- Checkout Cash atau QRIS
- Cetak dan cetak ulang struk thermal 58 mm
- Kirim struk ke WhatsApp customer tanpa menyimpan nomor pelanggan
- Template struk Daily Kopi ToGo dengan nomor transaksi otomatis
- Uang diterima dan kembalian otomatis untuk pembayaran Cash
- Upload dan tampilkan barcode QRIS milik toko saat pembayaran
- Laporan Daily, Weekly, Monthly
- Total transaksi, qty, omzet, laba
- Export seluruh penjualan, produk, dan laporan periode ke Excel
- Backup otomatis saat aplikasi dibuka, backup manual, dan restore data JSON
- Reset produk, transaksi, atau keranjang secara terpisah
- PIN Admin untuk melindungi reset data
- Menu Pengaturan untuk PIN Admin, backup, restore, export Excel, dan reset data
- Data tersimpan di browser menggunakan LocalStorage

Cara menjalankan:
1. Extract ZIP.
2. Buka file index.html di browser.
3. Aplikasi langsung bisa dipakai.

Catatan:
Aplikasi ini belum memakai database/server. Cocok untuk kasir sederhana dan latihan.
