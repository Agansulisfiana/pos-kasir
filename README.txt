POS Kasir Sederhana

Fitur:
- Dashboard ringkas seperti aplikasi POS
- Tambah produk
- Harga modal
- Harga jual
- Margin/laba otomatis
- Stok awal, terjual, stok akhir
- Transaksi kasir
- Laporan Daily, Weekly, Monthly
- Total transaksi, qty, omzet, laba
- Export produk dan laporan ke CSV
- Data tersimpan di browser menggunakan LocalStorage

Cara menjalankan:
1. Extract ZIP.
2. Buka file index.html di browser.
3. Aplikasi langsung bisa dipakai.

Catatan:
Aplikasi ini belum memakai database/server. Cocok untuk kasir sederhana dan latihan.
