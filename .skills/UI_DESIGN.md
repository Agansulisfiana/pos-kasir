# Skill: UI_DESIGN

## Purpose

Design every interface as a premium enterprise software website.

The website represents PT Reycom Integrated Solusi (PTRIS), a provider of enterprise identity, security, digital transformation, smart card, access control, and integrated technology solutions.

Every generated UI must reflect trust, professionalism, clarity, simplicity, and enterprise quality.

---

# Design Philosophy

Always prioritize:

- Professional
- Enterprise
- Clean
- Modern
- Spacious
- Trustworthy
- Consistent
- Accessible

Never create designs that resemble:

- Startup landing pages
- Crypto websites
- Gaming websites
- Colorful SaaS dashboards
- Cartoon style
- Over-decorated layouts

The overall appearance should be comparable to:

- HID Global
- Entrust
- Cisco
- Dell Technologies
- Lenovo Enterprise
- HP Enterprise
- Microsoft
- Bosch Security
- Zebra Technologies

# Consistency Rules

Every newly created component must match the existing design system.

Never introduce a different border radius.

Never introduce different shadows.

Never introduce different icon styles.

Never introduce inconsistent spacing.

Never use random fonts.

Never invent new colors.

---

# UI Review Checklist

Before generating any UI always verify:

✓ Consistent spacing

✓ Proper hierarchy

✓ Responsive

✓ Accessible

✓ Enterprise appearance

✓ Professional typography

✓ Consistent colors

✓ Correct icon style

✓ Premium card design

✓ Balanced whitespace

✓ Clean layout

✓ Soft animation

✓ No visual clutter

Only produce UI that would be acceptable for an enterprise technology company.
